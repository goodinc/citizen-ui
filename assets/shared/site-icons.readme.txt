
This file contains the original icons, from Adobe Illustrator:
icons.original.svg

It also contains the lowercase and uppercase alphabet of “Brandon Grotesque Bold” (a-z and A-Z) as a fallback for browsers that don’t support ligatures.


--
To add symbols to the font files (icons.ttf, icons.eot, etc…):

1. Visit this site:
http://icomoon.io/app/

2. Upload this file:
icons.json

3. Import your new icons

4. Download the new font files

Note: It may help if you begin by opening icons.original.svg in Adobe Illustrator and use it as a guide.
